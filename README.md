## 🧱 Working principles

- Build useful systems, not demos.
- Prefer small, reliable workflows over flashy automation that cannot be trusted.
- Keep humans in control of scope, approval, and judgment.
- Make agent work auditable: clear task ownership, durable handoffs, and visible outcomes.
- Treat AI-first operations as a company design problem, not just a tooling problem.
